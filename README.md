# SWE TH3: Wall visualizer
[Assessment](https://monumentalco.notion.site/SWE-TH3-Wall-visualizer-6d269ab0f1a342959a2ad02dc0b121d2) for [Monumental](https://www.monumental.co) by [Stan van Baarsen](https://www.stanvanbaarsen.nl).

November 8, 2025.

Tested using node v22.13.1, in Chrome v142.0, Safari v26.0.1 and Firefox 144.02

## 1. Install it
First, install the necessary project packages:
```
npm install
```

## 2. Launch it
Then, launch the web server in your terminal:
```
npm run dev
```

## 3. See it
Open the web app in your browser at http://localhost:5173/
